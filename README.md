# Bookstore Management System

Sistema de gerenciamento de livraria.

## 📖 Sobre a Aplicação
O objetivo deste projeto é fornecer uma plataforma para o controle de estoque de livros e organização por categorias. A aplicação permite realizar operações de CRUD (Criação, Leitura, Atualização e Exclusão), garantindo a persistência dos dados e a integridade das informações da livraria.

## 🚀 Tecnologias Utilizadas

O projeto foi construído utilizando um ecossistema moderno e escalável:

- **Linguagem:** [PHP 8.2+](https://www.php.net/)
- **Framework:** [Laravel 11](https://laravel.com/)
- **Banco de Dados:** [MySQL 8.0](https://www.mysql.com/)
- **Containerização:** [Docker](https://www.docker.com/) & [Docker Compose](https://docs.docker.com/compose/)
- **Gerenciador de Dependências:** [Composer](https://getcomposer.org/)
